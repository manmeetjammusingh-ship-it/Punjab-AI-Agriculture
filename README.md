# Punjab AI Agriculture

## Overview

This repository contains research materials, datasets, and analysis for my M.Sc. thesis:

**Techno-Economic Feasibility of Artificial Intelligence in Punjab's Crop Production**

## Objectives

- Assess the financial feasibility of AI in agriculture.
- Identify barriers to AI adoption among farmers.
- Evaluate the role of government interventions.

## Repository Structure

- 📁 data - Research datasets
- 📁 questionnaire - Survey questionnaire
- 📁 analysis - Statistical analysis
- 📁 docs - Documentation
- 📁 images - Figures and charts

## Tools Used

- Microsoft Excel
- Stata
- Python
